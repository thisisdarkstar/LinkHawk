# LinkHawk - Hunt URLs with Precision

LinkHawk is a powerful Chrome extension designed for bug bounty hunters and security researchers. Extract every URL from any webpage instantly with a single click.

## Features

- **One-Click URL Extraction** - Instantly crawl all URLs from any web page
- **Smart Filtering** - Search and filter extracted URLs by keyword
- **Pagination** - Navigate large URL sets with ease
- **Deduplication** - Automatic removal of duplicate URLs
- **Export Options** - Export results as TXT or JSON for further analysis
- **Dark/Light Theme** - Switch between themes with a single click
- **Cross-Platform** - Works with all popular bug bounty platforms

## Installation

1. Clone this repository or download as ZIP
2. Open Chrome and navigate to `chrome://extensions`
3. Enable **Developer mode** (toggle in top-right)
4. Click **Load unpacked** and select the extension directory
5. Pin LinkHawk to your toolbar for quick access

## Usage

1. Navigate to any webpage you want to crawl
2. Click the LinkHawk icon in your toolbar
3. Watch as all URLs are extracted instantly
4. Use the search bar to filter specific URLs
5. Adjust results per page using the selector
6. Click any URL to open it in a new tab
7. Export results as TXT or JSON for your workflow

## Permissions

- `activeTab` - Access the current tab to extract URLs
- `scripting` - Execute the crawler script on pages
- `*://*/*` - Run on all websites
